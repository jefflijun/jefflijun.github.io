Gartner发布新的数据分析平台报告

发表于 2017-03-11

Gartner2月中旬公布了2017年数据分析平台四象限图(Magic Quadrant for Advanced Analytics Platforms)。处于领先者(leader)象限的厂商有 IBM, SAS, RapidMiner和KNIME。

#值得注意的重要变化包括以下内容：

  1. 名称从Advanced Analytics Platforms变为
    Data Science Analytics。Gartner对新的名称Data Science Analytics给出的解释如下:”A cohesive software application that offers a mixture of basic building blocks essential for creating all kinds of data science solutions, and for incorporating those solutions into business processes, surrounding infrastructure and products.”。与Advanced Analytics Platforms，Gartner进一步强调对于各种数据分析方式的广泛支持及整合进业务流程和基础架构的能力。

  2. 在连续多年追赶后，IBM第一次超越SAS，成为leader象限的领跑者。

  3. 埃森哲(Accenture)等一批厂商因为不满足新的Data Science Analytics要求，被榜单移除。榜单又增加了 Dataiku, Domino Data Labs, H20.ai, MathWorks和Teradata。

  4. DELL把Quest卖出后，Quest从leader领域掉落到Challenger象限。

  5. Gartner在报告中着重提到，由于机器学习，人工智能和云计算的飞速发展， Amazon, Baidu和Google随时有可能推出相关的产品或服务，成为该领域的威胁者。其中Amazon和Google在此报告中被列为“honorable mention” list。

目前被列入榜单中的的厂商需要满足以下的基本要求：
– 2014-2015年度相关收入增长150%
– 至少200个付费的企业/机构客户
